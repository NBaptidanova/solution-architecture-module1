# Fit/Gap анализ

## Список систем

|Название|Функциональность|Комментарий|
|--------|----------------|-----------|
| ИС Алмазы | Присвоение идентификаторов  |  | 
|           | Отбор алмазов для цифрового аукциона (новая)       | Требуется доработка функционала подготовки ведомостей  |
|           | Планирование работ          | Добавить этап перевозки в/из лаборатории, этап оцифровки алмазов  в существующий алгоритм планирования    |
|           | Подготовка ведомостей       | Добавить функционал составления и ведения реестра алмазов, отобранных на аукцион       |
| ИС Аукцион | Публикация данных для клиентов (новая)| Нет возможности загружать и отображать аналитические отчеты об алмазах для онлайн аукциона. Требуется интеграция с облачным сервисом, настройка витрины данных. Требуется автоматическая интеграции с ИС Лаборатория для получения данных по алмазам по защищенному соединению с целью обеспечить целостность данных.  |
|			 | Проведение аукциона |   |
| ИС Бухгалтерия| Актирование и оплата услуг лаборатории (новая)  | Требуется доработка справочников, шаблонов документов для актирования услуг лаборатории.|
|               | Актирование и оплата услуг компаний-перевозчиков | |
|               | Ведение контрактов |  |
| ИС Логистика | Организация перевозки алмазов покупателям |  |
|              | Организация перевозки алмазов в лабораторию (новая)| Добавить этап доставки алмазов в лабораторию и обратно в офис в алгоритм планирования перевозок|
| ИС Лаборатория (новая, внешняя)| Ввод и хранение данных лабораторных исследований алмазов | Требуется автоматическая интеграция с ИС Аукцион для передачи данных о результатах исследований по защищенному каналу|
|                                | Актирование и оплата услуг лаборатории (новая)| Решение определяет владелец системы  | 


## Список интеграций

|Интеграция|Способ реализации|Передаваемые данные|
|----------|-----------------|-------------------|
|ИС Алмазы -> ИС Логистика (существующая)|автоматическая|Списки алмазов для перевозки, Данные адресатов, Плановые даты выполнения перевозки (существующие)|
|ИС Логистика -> ИС Бухгалтерия (существующая) |ручная| Акты выполненных работ, Счета на оплату (существующие)|
|ИС Лаборатория -> ИС Бухгалтерия (новая)|ручная|Акты выполненных работ, Счета на оплату (новые)|
|ИС Лаборатория -> ИС Аукцион (новая) |автоматическая| данные о результатах исследований алмазов (новые) |
|ИС Аукцион -> Облачный сервис с витриной данных (новая) |автоматическая|данные о результатах исследований алмазов (новые)|
